How to Verify

1. Click a test link above — it opens index.html with tracking params.

2. Submit the form on that page.

3. Check the resulting lead in PSS — the Referring Website field should contain the full URL with the tracking params (not just the plain page URL).

4. Use browser DevTools → Application → Session Storage to watch pss_referring_url_59 update in real time.
